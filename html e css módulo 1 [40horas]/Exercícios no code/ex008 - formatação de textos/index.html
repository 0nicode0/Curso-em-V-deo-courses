Texto deletado

Podemos marcar um texto como excluído par indicar que ele deve ser lido, mas não considerado. Com a tag del.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que foi adicionado depois, com a tag INS. (semântica).

Existe também o sublinhado com a tag U (não semântica).
